# Klix Ultra Launcher - Release v1.3

## New Features

### Visual & UI Enhancements
- **Animated Background**: Added a stunning new dynamic background with gradients and constellation particle effects.
- **Custom Title Bar**: Replaced the standard Windows frame with a sleek, custom-designed title bar that integrates perfectly with the dark theme.
- **Smooth Animations**: Introduced fade transitions between pages and animated hover effects for all buttons and toggles.
- **Refined Interface**: Improved transparency effects, padding, and font sizes across the application for a more modern look.

### Performance & Core
- **Smart Caching System**: Implemented a robust caching manager for API requests and images, significantly speeding up the browsing experience.
- **Optimized Launching**: Fixed a critical issue where game files were re-downloaded on every launch. The launcher now intelligently verifies existing files.
- **Auto-Save Settings**: Removed the manual save button. Settings are now saved automatically with a visual "Saved" indicator.

### Quality of Life
- **Clear Cache**: Added a dedicated button in settings to clear temporary files and free up space.
- **Compact Sidebar**: Optimized the layout of the installed content sidebar for better space utilization.

## Bug Fixes
- Fixed a crash related to account loading (`token` error) when using older account files.
- Resolved graphical glitches and console spam related to page transition animations.
- Fixed transparency issues where the background wouldn't show through the changelog viewer.
- General stability improvements and code cleanup.

# Klix Ultra Launcher - Release v1.2

## New Features

### Modern UI Overhaul
- Completely redesigned interface with a sleek, dark theme.
- New top-bar navigation for easier access to Mods, Resource Packs, and Shaders.
- Integrated skin preview in the main dashboard.

### Integrated Content Browser
- **Modrinth Integration**: Browse, search, and install Mods, Resource Packs, Shaders, and Modpacks directly from the launcher.
- **Smart Filtering**: Sort by relevance, downloads, follows, or update date.
- **One-Click Install**: Automatically resolves dependencies and installs the correct version for your loader.

### Advanced Version Management
- Create custom profiles with specific Minecraft versions and Mod Loaders (Fabric, Forge, Quilt, Vanilla).
- Separate directories for each profile to keep your mods and configs organized.
- Easy duplication and editing of existing profiles.

### Performance & Core
- **Auto Java Management**: Automatically detects, downloads, and installs the correct Java runtime for your selected Minecraft version.
- **Optimized Launching**: Faster game startup with detailed log output for debugging.
- **System Tray Support**: Option to minimize the launcher to the system tray to keep it running in the background.

## Bug Fixes
- Fixed an issue where the launcher would not close properly.
- Resolved crashes related to log window cursor positioning.
- Improved handling of large file downloads.
- Fixed display names for installed mods to show readable titles instead of filenames.

## Localization
- Full support for English and Arabic languages.
